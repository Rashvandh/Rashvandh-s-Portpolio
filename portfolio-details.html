AI Resume Analyzer & Job Recommendation System

This project is a full-stack web application designed to help job seekers improve their resumes and discover relevant job opportunities. The system leverages Natural Language Processing (NLP) and Machine Learning to analyze resume content and recommend best-fit job roles.

Key features include AI-driven resume parsing, skill extraction, job-role matching, and an intuitive dashboard. Built with HTML, CSS, JavaScript (frontend), and Java (backend), it offers a seamless user experience and real-time results.
